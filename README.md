Customer Survival Analysis & Churn Prediction

Overview

Customer churn is one of the most critical challenges for subscription-based and service-driven businesses. This project focuses on **predicting customer churn and analyzing survival patterns** using machine learning and statistical techniques to support **data-driven retention strategies**.

The system combines **Exploratory Data Analysis (EDA), Survival Analysis, and Predictive Modeling** to identify at-risk customers and uncover key factors influencing churn.

Objectives

* Predict customer churn using machine learning models
* Analyze customer survival probability over time
* Identify key drivers influencing churn behavior
* Provide actionable insights for customer retention strategies

Key Features

Data preprocessing and feature engineering
Exploratory Data Analysis (EDA) with visual insights
Survival analysis using Kaplan-Meier estimation
Machine Learning models for churn prediction
Model evaluation and performance comparison
Insight generation for business decision-making

Tech Stack

* **Programming:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn
* **Visualization:** Matplotlib, Seaborn
* **Statistical Analysis:** Lifelines (Survival Analysis)
* **Database (optional):** SQL

Survival Analysis

* Applied **Kaplan-Meier estimator** to analyze customer retention over time
* Estimated survival probability and churn risk at different time intervals
* Identified segments with higher churn likelihood

* Decision Tree
