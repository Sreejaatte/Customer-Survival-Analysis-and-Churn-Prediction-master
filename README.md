# 📊 Customer Survival Analysis & Churn Prediction

## 🚀 Overview

Customer churn directly impacts revenue and growth. This project combines **machine learning and survival analysis** to predict churn and estimate customer retention over time, enabling businesses to take **proactive retention actions**.

---

## 🎯 Objectives

* Predict customer churn using machine learning models
* Analyze customer survival probability over time
* Identify key factors influencing churn behavior

---

## 📂 Dataset

* Source: Kaggle (Telco Customer Churn Dataset)
* Description: Customer demographics, usage behavior, and churn labels

---

## 🔍 Exploratory Data Analysis (EDA)

* Identified key features influencing churn (tenure, contract type, usage patterns)
* Detected class imbalance in churn distribution
* Visualized trends using correlation heatmaps and distribution plots

---

## 🤖 Model Development

* Algorithms used: Logistic Regression, Random Forest
* Feature engineering: Encoding categorical variables, scaling numerical features
* Survival analysis: Kaplan-Meier estimator to model retention probability

---

## 📊 Evaluation Metrics

* Accuracy
* Precision / Recall
* F1 Score
* ROC-AUC

---

## 📈 Results & Insights

* Identified high-risk customer segments based on tenure and contract type
* Random Forest provided better predictive performance
* Survival curves highlighted time periods with highest churn probability

---

## 💡 Business Impact

* Enables targeted customer retention strategies
* Helps reduce churn and improve customer lifetime value
* Supports data-driven decision-making for marketing and product teams

---
